# Fluent Frontend

Фронтенд-часть мессенджера **Fluent** на Next.js с поддержкой реального времени через Socket.IO и интернационализацией через next-intl.

---

## Стек

| | |
|---|---|
| **Next.js + Bun** | фреймворк и рантайм |
| **Tailwind CSS** | стилизация |
| **RTK + RTK Query** | стейт-менеджер и работа с API |
| **Socket.IO** | события реального времени |
| **next-intl** | интернационализация (i18n) |
| **Vitest + RTL** | unit и UI тесты |
| **Storybook** | скриншот-тесты компонентов |
| **Nginx + Docker** | веб-сервер и контейнеризация |

---

## Архитектура

Проект построен по методологии **Feature-Sliced Design (FSD)**.

```
src/
├── app/           # инициализация, провайдеры, роутинг
├── widgets/       # крупные самостоятельные блоки UI
├── features/      # фичи с бизнес-логикой
├── entities/      # бизнес-сущности (user, chat, message...)
└── shared/        # UI-кит, утилиты, конфиги, хуки
```

---

## Запуск

```bash
bun install
bun dev
```

Открой [http://localhost:3000](http://localhost:3000) в браузере.
